Motu README:
=========================

To build the binaries from the source code, the following maven repository can be added http://mvnrepo-ext.cls.fr/nexus/content/repositories/cls-to-ext.

This maven repository include all the 3rd party artifacts used by Motu and not available on Maven Central (http://repo1.maven.org/maven2/). 

To add this maven repository choose one of the following methods:

- modify the maven setting.xml
- add it on an entreprise maven manager
- uncomment it in motu-parent/pom.xml
